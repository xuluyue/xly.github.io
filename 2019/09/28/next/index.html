媒体查询：

1、什么是媒体查询？

媒体查询是css3新语法

  • 使用媒体查询可以针对不同的媒体类型定义不同的样式（常见的 就是针对屏幕大小的变化而改变不同的css样式）
  • @media可以针对不同的屏幕尺寸设置不同的样式
  • 当你设置浏览器大小的过程中，页面也会根据浏览的宽度和高度重新渲染页面
  • 主要针对手机、平板等使用媒体查询

语法规范：

@media mediatype and|not|only （media fea{

​ css code;

}

  • 用media开头

  • mediatype 媒体类型

    常见值有：

    all ：用于所有设备

    print ： 用于打印机和打印预览

    screen ：用于电脑屏幕，手机，平板等

  • 关键字and|not|only

  • media feature 必须有小括号

    值：

    width：页面可见区域的宽度

    min-width:页面最小可见宽度

    max-width:页面最大可见区域宽度

    ​

例:

在我们的屏幕上并且最大的宽度是800像素 设置我们想要的样式

@media screen and （max-width：800px）{

​ background-color:pink;

}当屏幕的宽度小于等于800像素时，背景颜色时粉色

@media screen and （max-width：500px）{

​ background-color:blue;

}``当屏幕的宽度小于等于500像素时，背景颜色时蓝色`

2、媒体查询+rem实现元素动态大小变化

通过媒体查询，给不同的页面大小设置不同的html字体大小，再通过rem就此改变元素的大小

例：

@media screen and （min-width：320px）{

​ html{

​ font-size:50px;

​ }

}//当屏幕大小最小为320px时，html的字体大小为50px

@media screen and （min-width：600px）{

​ html{

​ font-size:100px;

​ }

}当屏幕大小最小为600px时，html的字体大小为100px
